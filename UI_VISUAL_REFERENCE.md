## State Variations

### **Button States**
```
DEFAULT:    Sky Blue (#1E90FF), White text
HOVER:      Darker sky blue (#1873CC), Scale 105%
ACTIVE:     Darkest shade, Scale 95%
DISABLED:   Gray (#D1D5DB), Opacity 50%
```

### **Input States**
```
DEFAULT:    White bg, Sky blue border (2px)
FOCUS:      Sky blue border (2px), Blue ring
DISABLED:   Gray bg, Gray border, Reduced opacity
ERROR:      Red border, Light red background
```

### **Card States**
```
DEFAULT:    White bg, Light shadow, Blue border
HOVER:      Slightly darker shadow
ACTIVE:     Blue ring highlight
```
