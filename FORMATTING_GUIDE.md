# Question Paper Formatting Guide

## Overview
The LaTeX question papers generated by this application now follow professional examination paper standards with exact formatting, alignment, spacing, and layout.

## Key Formatting Features

### 1. **Professional Header Design**
- **Boxed Title**: Institution/exam name in bordered box
- **Exam Details Table**: Duration, maximum marks, difficulty level, and date
- **Header & Footer**: Subject name in header, page number, and "Turn over" in footer
- **Top Border**: 2pt thick rule for visual impact

### 2. **Instruction Box**
- **Centered Title**: "GENERAL INSTRUCTIONS" in large bold
- **Numbered List**: Clear, spaced instructions
- **Section Breakdown**: Automatically includes question distribution
- **Professional Box**: Proper padding and border

### 3. **Question Layout**

#### MCQ Format:
```latex
Q.1                                                              [1 mark]

Question text here...

(a) Option A
(b) Option B
(c) Option C
(d) Option D

Solution:
Detailed explanation...
```

#### Short Answer Format:
```latex
Q.3                                                              [2 marks]

Question text here...

Solution:
Step-by-step solution...
```

#### Long Answer with Sub-parts:
```latex
Q.5                                                              [5 marks]

Main question text...
(a) Sub-question 1                                          [2 marks]
(b) Sub-question 2                                          [2 marks]
(c) Sub-question 3                                          [1 mark]

Solution:
(a) Solution for part a...
(b) Solution for part b...
(c) Solution for part c...
```

#### Fill in the Blanks:
```latex
Q.4                                                              [1 mark]

Fill in the blank: The value is ___________ units.

Solution:
The correct answer is...
```

### 4. **Section Organization**
- **Section Headers**: Large, centered, with descriptive text
- **Section Dividers**: Horizontal rules between sections
- **Mark Distribution**: Clearly indicated for each section

### 5. **Spacing Standards**
- **Line Spacing**: 1.15 for better readability
- **Between Questions**: 0.5cm with divider rule
- **Section Spacing**: 0.4cm after section headers
- **Paragraph Spacing**: Consistent throughout

### 6. **Margins & Layout**
- **Paper Size**: A4 (210mm × 297mm)
- **Left Margin**: 2cm
- **Right Margin**: 2cm
- **Top Margin**: 2.5cm
- **Bottom Margin**: 2.5cm

### 7. **Typography**
- **Base Font**: 12pt
- **Question Numbers**: Bold
- **Marks**: Bold, right-aligned
- **Solutions**: Bold "Solution:" label
- **Section Titles**: Large, bold, centered

## Pattern Matching Feature

When you upload a sample question paper pattern, the system will:

1. **Analyze** the exact structure, formatting, and layout
2. **Extract** header format, instruction text, section structure
3. **Replicate** question numbering, mark indication, spacing
4. **Match** MCQ format, fill-in-blank style, visual elements
5. **Generate** new questions that fit the exact pattern

### What Gets Replicated:
✅ Header format and title styling
✅ Institution name and exam details
✅ Instructions (word-for-word)
✅ Section names and structure
✅ Question numbering style (Q.1, 1., Question 1, etc.)
✅ Mark indication format ([5], (5 marks), [5M], etc.)
✅ MCQ option format (a/b/c/d vs A/B/C/D vs 1/2/3/4)
✅ Spacing, boxes, lines, and visual elements
✅ Page layout and margins
✅ Footer and page numbering style

### What Changes:
❌ Actual question content (new questions from your textbook)

## Question Type Examples

### 1. Multiple Choice Questions (MCQ)
- Options labeled as (a), (b), (c), (d)
- Left margin: 1.5cm
- Item spacing: 2pt
- Solution indicates correct option with explanation

### 2. Fill in the Blanks
- Blank lines: `\underline{\hspace{3cm}}`
- Proper length based on expected answer
- Clean, professional appearance

### 3. True/False
- Clear T/F options
- Brief justification in solution

### 4. Short Answer (2-3 marks)
- Concise question
- Step-by-step solution
- Proper mathematical notation

### 5. Long Answer (5-10 marks)
- May have sub-parts
- Detailed solutions for each part
- Mark breakdown shown

## Mathematical Notation

All mathematical expressions use proper LaTeX:
- **Inline math**: `$x + y = 5$`
- **Display math**: `\[x^2 + y^2 = r^2\]`
- **Fractions**: `$\frac{a}{b}$`
- **Square roots**: `$\sqrt{x}$`
- **Superscripts**: `$x^2$`
- **Subscripts**: `$a_1$`
