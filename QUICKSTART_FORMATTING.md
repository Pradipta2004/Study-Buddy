## Troubleshooting

### "Pattern not matching"
- Ensure pattern PDF is text-based, not scanned
- Try a different pattern paper
- Check if PDF is password-protected

### "Questions look different"
- Default format applies when no pattern uploaded
- Upload pattern for exact matching
- Verify pattern PDF uploaded correctly

### "LaTeX compilation error"
- Download LaTeX file and check manually
- Special characters might need escaping
- Report issue with error log
